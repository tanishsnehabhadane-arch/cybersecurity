# Cyber Security Bot

A simple cybersecurity analysis tool built using **Python**, **Streamlit**, and security intelligence APIs. This project allows users to analyze domains and gather security-related information through an easy-to-use web interface.

## Overview

The goal of this project is to provide a lightweight cybersecurity assistant that can perform basic domain intelligence gathering and display the results in a clean dashboard.

This project was created as a learning exercise to understand:

* API integration
* Cybersecurity intelligence gathering
* Streamlit web applications
* Data visualization and reporting

## Features

* Domain reputation analysis
* Security information lookup
* User-friendly Streamlit interface
* Real-time API requests
* Easy deployment and setup

## Technologies Used

* Python
* Streamlit
* Requests
* Cybersecurity Intelligence APIs

## Installation

Clone the repository:

```bash
git clone https://github.com/tanishsnehabhadane-arch/cybersecurity.git
```

Move into the project directory:

```bash
cd cybersecurity
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## Usage
1. Launch the application.
2. Enter a domain name.
3. Click **Analyze**.
4. Review the returned security information.
